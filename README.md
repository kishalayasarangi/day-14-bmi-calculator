# Day 14 — BMI Calculator

**120 Days of Code** | NIT Rourkela | Mechanical Engineering

## What this is
A BMI calculator with health tips, ideal weight range,
and a visual BMI scale bar.

## How to view
Open `index.html` in your browser.

## Features
- Metric and Imperial unit support
- Visual BMI scale with marker
- Health tips based on result
- Ideal weight range calculator
- Gender and age input
